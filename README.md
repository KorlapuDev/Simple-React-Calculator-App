#### Problem statement:

#### Technologies Used:

- HTML
- CSS
- JavaScript
- React Js

## Install & Run Project

To install all the dependences of the project, run the following command:

    git clone https://github.com/KorlapuDev/Simple-React-Calculator-App.git
    yarn install
    yarn start
